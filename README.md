# 📄 Formulario Culturama

Este projeto consiste na criação de um formulário web chamado Formulário Culturama, desenvolvido com foco em boas práticas de HTML, CSS, acessibilidade e organização de código.

---

### 🚀 Objetivo

O objetivo do projeto foi praticar a construção de formulários modernos e acessíveis, aplicando conceitos importantes do desenvolvimento front-end.

---

### 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- Git e GitHub

---

### 📚 O que foi aplicado
Uso de diferentes tipos de inputs:
- text
- email
- checkbox
- radio
- select
  
Estruturação semântica com:
- fieldset
- legend
- label

Uso da tag:
- textarea para campos de texto maiores
- Boas práticas de acessibilidade
- Organização visual com CSS
- Versionamento de código com Git (branches, commits e merge)

---

### 🎯 Funcionalidades
- Formulário organizado e intuitivo
- Melhor experiência para o usuário
